Jeu des données

Le jeu des données choisi c'est le Données sur les réseaux mobiles de l'Agence Nationale des Fréquences (ANFR) sur le format CSV (chaque données separé par point-virgule).

Explication du site

Pour la création de les pages le script utilise les lignes des codes html présentent dans le fichier code.html et style.css pour les designer.

Dans la page suivante, vous allez retrouver les bilans des antennes-relais des réseaux mobiles divisés par catégories : l’opérateur, la technologie, le système et le statut de l’installation.

Pour chaque élément de la catégorie opérateur, il y a une page pour la présentation des ses données.

Nouvelle valeur calculée

Dans le jeu des données on a utilisé 6 colonnes : nom du opérateur (c. 2), type de technologie (c. 4), date de création (c. 5), génération (c. 8), coordonnées (c. 21) et statut de l’installation (c. 22). On a calculé les total des antennes-relais et les filtrées selon les catégories au-dessus.
